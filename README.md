# design3report
Latex report for design 3

Made with Latex :).

## Installation

1. `sudo apt install texlive-latex-extra -y`
2. `sudo apt-get install texlive-bibtex-extra biber -y`

## Compile

1. `pdflatex main`
2. `biber main`
3. `pdflatex main`

## Git
